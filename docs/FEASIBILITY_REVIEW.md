## ⚠️ 실현 가능성 재검토 (냉정한 평가)

### ✅ 즉시 가능 (기술적 장벽 없음)

| 피처 | 근거 | 리스크 |
|---|---|---|
| #1 자연어 공지 파서 | 텍스트 파싱은 LLM 최적 업무 | 선사 사이트 구조 변경 시 재조정 필요 |
| #2 치어 방류 예측 | 공공데이터포털 데이터 존재 확인 | 방류↔조과 상관관계 검증 필요 (1~2시즌) |
| #3 SNS 바이럴 모니터링 | 블로그/유튜브 크롤링 기술 보편화 | 네이버 밴드/카페 비공개 그룹 접근 불가 |

| #7 전문가 챗봇 | RAG+페르소나 프롬프트로 구현 | 초기 지식 DB 구축에 2~4주 소요 |
| #8 법규 QA | RAG 최적 유스케이스 | 법규 변경 시 벡터 DB 업데이트 필요 |
| #9 음성 기록 | Web Speech API (브라우저 내장) | iOS Safari 호환성 테스트 필요 |

## 💵 수익 vs 비용 시뮬레이션

| 시나리오 | 유저 수 | 월 비용 | 월 매출 (예상) | 손익 |
|---|---|---|---|---|
| **Phase 1** (런칭) | 1,000명 | ~₩5,000 | ₩0 (무료) | -₩5,000 |
| **Phase 2** (성장) | 5,000명 | ~₩25,000 | 프로회원 50명 × ₩4,900 = ₩245,000 | **+₩220,000** |
| **Phase 3** (안정) | 10,000명 | ~₩50,000 | 프로 200명 + 마스터 50명 = ₩1,475,000 | **+₩1,425,000** |
| **Phase 4** (확장) | 50,000명 | ~₩200,000 | 프로 1,000명 + 커머스 수수료 | **+₩7,000,000+** |

> [!TIP]
> **손익분기점**: 프로회원 **11명** (₩4,900 × 11 = ₩53,900) 이면 전체 LLM 비용 커버.
> **즉, 유저 11명만 프로 구독하면 흑자.**
