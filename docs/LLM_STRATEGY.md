## 🎯 LLM 킬러 피처 전체 로드맵 (10개로 확장)

### 🥇 Tier 1: 핵심 차별화 (선점 필수)

#### 1. 자연어 공지 파서 (Natural Language Notice Parser)
- 수백 선사의 비정형 공지 → `{선사, 어종, 예약오픈일, 잔여석}` 구조화
- **비용**: Gemini Flash ~₩500/월

#### 2. 치어 방류 기반 시즌 예측 AI 🆕
> **"올해 오천항 주꾸미, 44만 마리 방류 → 작년 대비 2배 → 풍작 예상"**

| 인풋 데이터 | 출처 | 수집 방법 |
|---|---|---|
| 지역별 방류량 | 공공데이터포털 + 뉴스 | 파일다운로드 + 크롤링 |
| 과거 해당 지역 어획량 | FishLog 조과 데이터 | 자체 DB |
| 수온 변화 | 기상청 해수면 온도 API | API |
| 해류 패턴 | 국립해양조사원 | API |

**LLM 역할**: 다차원 데이터를 종합하여 자연어 시즌 프리뷰 생성
```
📊 2026 주꾸미 시즌 프리뷰 (오천항)
━━━━━━━━━━━━━━━━━━━━
🐙 방류량: 44만 마리 (전년 대비 +120%)
🌊 수온 예측: 평년 수준 (적정)
📈 AI 예상: ★★★★☆ 풍작 가능성 높음
⏰ 예약 전쟁 예상 강도: 🔴 매우 격렬
💡 팁: 2월부터 예약 오픈 감시 추천
```

#### 3. SNS 사전 시그널 & 바이럴 채비 모니터링 🆕
> **금어기 해제 전 분위기 파악 + 지금 제일 핫한 채비 캐치**

| SNS 채널 | 모니터링 대상 | LLM 역할 |
|---|---|---|
| **유튜브** | "주꾸미 꿀팁" 관련 최신 영상 | 품절 대란템, 조회수 급증 채비 추출 |
| **블로그/카페** | "야마시타 vs 요즈리 vs 국산" 리뷰 | 실시간 유행 채비 랭킹화 |
| **인스타그램** | 선장님 조과 인증 사진 | 당일 가장 잘 먹힌 에기 색상 포착 |

**💰 커머스 연계 (수익화)**:
- 야마시타 틴셀, 요즈리 슷테 등 **실시간 바이럴 타는 채비/장비를 즉시 감지**
- 품절 대란이 일어나기 전, 유저에게 "지금 입소문 타는 채비" 알림 발송
- 앱 내 **커머스 스토어(또는 제휴몰) 바로가기/결제 버튼** 제공 → 쇼핑 수수료 수익률 극대화

**비용**: 네이버 검색 API(무료) + Gemini Flash 감성/키워드 분석 ~₩300/월

#### 4. 개인 맞춤 오픈런 알림 (Smart Alert)
- 유저 조과 기록 분석 → 선호 어종/지역 자동 감지
- 맞춤 알림 구독 → "오천항 주꾸미 예약 오픈되면 알려줘"

#### 5. GPS 기반 스마트 전자승선명부 🆕
> **"항구에 도착하면 자동으로 승선명부 작성 끝"**

| 구현 원리 | 기술 / API |
|---|---|
| **사전 작성** | FishLog 앱에 유저 본인/일행 정보 미리 저장 |
| **자동 체크인** | 예약 당일, 선착장 반경 50m 진입 시 GPS 지오펜싱(Geofencing)이 자동 감지 |
| **선장 확인** | 선장용 대시보드에 "탑승 완료"로 자동 전환 |
| **해경 제출** | 해양수산부 '공동활용체계 API' 연동 → 해경에 1클릭 자동 전송 |

**조사 결과**: 해양수산부 '낚시해(海)' 시스템은 이미 민간 낚시 앱과의 출입항신고 연계(Open API)를 지원함. **법적/기술적 타당성 100% 확보.**
**기대 효과**: 새벽 4시에 칠흑 같은 배 위에서 종이 명부에 볼펜으로 가족들 주민번호를 적어야 하는 최악의 Pain Point 완벽 해결.

#### 6. AI 낚시 컨시어지 ("요즘 뭐 하러 가?") 🆕
> **한 마디만 물어보면, 출조 A to Z 완성 코스가 나온다**

유저가 "요즘 어떤 낚시 가야 해?"라고 물어보면 → LLM이 **실시간 트렌드 + 유저 프로필 + 현재 시즌**을 종합하여 완전한 출조 플랜을 자동 생성.

**인풋 데이터 종합**:
| 데이터 | 출처 | 용도 |
|---|---|---|
| 시즌 인기 어종/지역 | FishLog 조과 DB + SNS 트렌드 | "요즘 뭐가 핫한지" |
| 유저 경험치 | 가입 시 입력 (초보/중급/고급) | 난이도 맞춤 |
| 유저 위치 | GPS / 거주 지역 | 가까운 항구 우선 배정 및 소요 시간 계산 |
| 교통/출발 | 유저 선택 (자차/버스) + 실시간 내비 | 출항 시간 기준 출발/기상 시간 역산 알림 |
| 예산 | 유저 선택 | 장비 대여 vs 구매 추천 |

**출력 예시** (초보 유저, 서울 강남 기준, 자차 이용):
```
🎣 AI 추천: 9월 주꾸미 선상낚시 코스
━━━━━━━━━━━━━━━━━━━━━━━━━━
📈 요즘 인기: 주꾸미 (금어기 해제 직후, 전국 1위)
📍 추천 지역: 오천항 (조과 최상위)
🚗 교통/이동: 자차 이동 (강남 → 오천항, 실시간 예상 2시간 15분 소요)
     └ ⏰ 추천 기상/출발 알람 설정: 새벽 2시 15분 출발
        (출항 1시간 전인 새벽 4시 30분 선착장 도착 권장)
💰 예산: ~8만원 (승선비 6만 + 장비 대여 2만)
🎒 준비물 체크리스트:
     ☐ 신분증 (실물 없어도 OK! PASS앱, 모바일 신분증 가능)
     ☐ 멀미약 (전날 밤 1알 + 승선 1시간 전 1알)
     ☐ 버릴 각오한 어두운 옷 (먹물 테러 대비)
     ☐ 다이소 지퍼백 (잡은 주꾸미 담아갈 용도)

💡 꾸미 교수의 [선제적 꿀팁] (초보 필수 시크릿)
  - 🦑 먹물 지우는 법: 과탄산소다 푼 물에 담그면 끝!
  - 🎣 조과 차이 비밀: 초보는 바닥 찍는 감을 몰라서
    고수와 10배 차이도 납니다. 선장님께 먼저 물어보세요!
  - 💺 자리 팁: 멀미가 심하면 무조건 '선미(뒤쪽)' 사수!

⚠️ 승선명부: FishLog 앱으로 자동 제출 (터치 불필요)
━━━━━━━━━━━━━━━━━━━━━━━━━━
[📱 해당 코스로 선사 예약] [🎣 장비 세트 구매/대여]
```

**핵심 차별화 (Proactive AI)**: 경쟁 앱은 능동적 검색을 요구하지만, FishLog는 **"초보가 몰라서 못 묻는 질문(먹물 지우는 법, 멀미약 골든타임, 신분증 지참)까지 선제적으로(Proactive) 알려주는"** 압도적 편의성을 제공합니다.

### 🥈 Tier 2: 핵심 부가 가치

#### 5. 실시간 금어기 해제일 조황 대시보드 🆕
> **9/1 당일, 전국 조황을 실시간 종합**

| 데이터 소스 | 정보 |
|---|---|
| 뉴스 기사 (자동 크롤링) | "오천항 주꾸미 평균 15마리" |
| 선사 조황 게시판 (크롤링) | 각 선사별 실시간 조과 |
| 유저 제보 (앱 내) | "지금 여기 대박" 실시간 공유 |
| 유튜브 라이브 (크롤링) | 실시간 방송 중 조황 정보 |

**LLM 역할**: 여러 소스의 비정형 조황 정보를 종합하여 지역별 실시간 요약
```
🔴 LIVE 9/1 금어기 해제 조황 (15:30 기준)
━━━━━━━━━━━━━━━━━━━━
🏆 1위 오천항: 평균 18마리 (🔥 대박)
🥈 2위 대천항: 평균 12마리 (👍 양호)  
🥉 3위 군산항: 평균 8마리 (보통)
📉 인천 남항: 평균 5마리 (기대 이하)
💡 AI 분석: 수온 22℃ 이상 지역이 호조황
```

#### 6. 물때+날씨 기반 예상 조과 예측 🆕
> 해외 Fishing Forecast(TipTop) 앱 한국판

| 인풋 | 출처 |
|---|---|
| 물때 (조금/사리) | 국립해양조사원 API |
| 날씨 (풍속/운량) | 기상청 API |
| 수온 | 기상청 해수면 온도 |
| 달의 위상 | 천문 데이터 |
| 과거 조과 패턴 | FishLog 자체 DB |

**LLM 역할**: 복합 데이터 → 자연어 예측 (단, **"AI 예측, 참고용" 면책 필수**)

#### 7. 🎣 어종별 전문가 페르소나 챗봇 ("프로 낚시 비서") 🆕
> **"고수만 아는 정보를, AI가 누구에게나"**

각 어종/장르마다 **전문가 페르소나**가 배치. 입문자도 고급 노하우 즉시 습득.

| 페르소나 | 어종 | 말투 | 전문 지식 |
|---|---|---|---|
| 🐙 **꾸미 교수** | 주꾸미/갑오징어 | 꼼꼼한 학자 | 에기 색상×날씨, 물때별 포인트, 시즌 전략 |
| 🐟 **우럭 선장** | 우럭/볼락 | 호탕한 어부 | 바닥층 공략, 채비법, 밑걸림 대처 |
| 🎣 **참돔 마스터** | 참돔/감성돔 | 장인 스타일 | 타이라바/인치쿠, 조류 읽기 |
| 🦑 **한치 헌터** | 한치/오징어 | 야행성 달인 | 야간 채비, 집어등, 수온별 수심 |
| 🐠 **광어 프로** | 광어/넙치 | 과학적 분석가 | 바운싱, 웜 선택, 바닥 지형 |
| 🌊 **만능 조사** | 종합/초보 | 친절한 형/누나 | 입문 가이드, 멀미, 안전, 에티켓 |

**장비 추천 3단계** (예: 우럭 선장이 답변)
```
📦 입문 (₩15~25만) — 카고채비 5:5로드 + 베이트릴
📦 중급 (₩40~60만) — 카본로드 + 전동릴 (수심50m↑)
📦 고급 (₩100만+) — 티타늄팁 전용로드 + 시마노 전동릴
💡 오늘 오천항 수심 40m → 전동릴 추천
```

**RAG 아키텍처**: 유튜브 고수 영상 요약 + 커뮤니티 꿀팁 + FishLog 유저 조과 → 벡터 DB

**수익화**:
| 등급 | 범위 | 가격 |
|---|---|---|
| 무료 | 일 3회 질문 + 기본 추천 | ₩0 |
| 프로 | 무제한 + 포인트 추천 | ₩4,900/월 |
| 마스터 | 프로 + 1:1 코칭 + 시즌 리포트 | ₩9,900/월 |

## 낚시인 11대 페인포인트 & 솔루션 매핑

| # | 페인포인트 | 현재 상태 | FishLog 솔루션 | 피처 # |
|---|---|---|---|---|
| 1 | 어떤 선사가 예약 언제 여는지 모름 | 일일이 확인 | 자연어 공지 파서 (LLM) | #1 |
| 2 | 인기 선사 예약 수분 내 마감 | 늦으면 끝 | 맞춤 알림 즉시 푸시 | #4 |
| 3 | 새벽 배 위에서 종이 승선명부 작성 | 수기 작성 고통 | **GPS 자동 체크인 전자승선명부** | **#5** |

## 가드레일 원칙 (전체 공통)

| 원칙 | 구현 |
|---|---|
| **날짜/예약 = DB only** | LLM이 날짜를 추론/생성하지 않음. DB 조회 결과만 표시 |
| **신뢰도 점수** | LLM 추출 결과에 정규식 매칭 기반 신뢰도 부여 |
| **출처 표시** | 모든 AI 답변에 원본 링크/데이터 출처 명시 |
| **면책 문구** | 예측/추천에 "AI 참고용, 실제와 다를 수 있음" 표시 |
| **Circuit Breaker** | LLM API 장애 시 → 기본 DB 검색 폴백 |
| **Rate Limiting** | 유저당 일 50건 제한 → 비용 폭주 방지 |

## 최종 결론

경쟁사 11개 중 LLM을 활용하는 앱은 **0개**. 선점하면 최소 1~2년 독점 가능.
